# Thesis Modular Chapter Layout

This folder contains section-wise editable chapter content.

- `ch1/`: Introduction sections
- `ch2/`: Related Work sections
- `ch3/`: Methodology and System Design sections
- `ch4/`: Implementation and Results sections
- `ch5/`: Conclusion and Future Work sections

Main chapter wrapper files remain:
- `text/C1.tex` through `text/C5.tex`

Edit section content inside the `text/chapters/ch*/` files.
