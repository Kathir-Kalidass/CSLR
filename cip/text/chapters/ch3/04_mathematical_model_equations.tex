\section{Mathematical Model / Equations}
There are many criteria that can be used to evaluate the performance of a sign language recognition model. For isolated sign recognition, the commonly used metric is accuracy. Precision, recall, and F1-score are also widely used as complementary metrics. The performance metrics used in this study are given in Equations (1)--(4).

\begin{equation}
\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}
\end{equation}

\begin{equation}
\text{Precision} = \frac{TP}{TP + FP}
\end{equation}

\begin{equation}
\text{Recall} = \frac{TP}{TP + FN}
\end{equation}

\begin{equation}
F_1\text{ Score} = \frac{2TP}{2TP + FP + FN}
\end{equation}

where $TP$ is true positive, $TN$ is true negative, $FP$ is false positive, and $FN$ is false negative.
