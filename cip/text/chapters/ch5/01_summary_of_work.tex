\section{Summary of the Work}
This project presented the design and implementation of a real-time Indian Sign Language (ISL) recognition system using deep learning techniques. The proposed system processes RGB video input and extracts both spatial and temporal features to accurately recognize hand gestures.

Spatial features are extracted using EfficientNet, while temporal dependencies are captured using a CNN--RNN architecture. A multi-feature attention mechanism is incorporated to improve feature selection by focusing on relevant gesture components. The system further uses a Transformer model for sequence modeling and a CTC layer for temporal alignment. The final output is generated in both text and speech form, enabling effective communication between sign language users and non-signers.
