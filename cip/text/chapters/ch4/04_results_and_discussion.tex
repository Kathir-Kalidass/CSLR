\section{Results and Discussion}
The proposed system successfully recognizes Indian Sign Language gestures using a combination of spatial and temporal features.

\subsection{Key Observations}
\begin{itemize}[leftmargin=1.5em]
\item EfficientNet effectively captures spatial features such as hand shape.
\item CNN--RNN captures motion dynamics across frames.
\item Attention mechanism improves feature selection and accuracy.
\item Transformer enhances sequence understanding.
\end{itemize}

\subsection{Performance Metrics}
The system is evaluated using:
\begin{itemize}[leftmargin=1.5em]
\item Accuracy,
\item Precision,
\item Recall,
\item F1-score.
\end{itemize}

The model achieves good recognition accuracy for isolated gestures while maintaining computational efficiency.

\subsection{Intermediate Outputs}
The system produces several intermediate outputs:
\begin{itemize}[leftmargin=1.5em]
\item Extracted frames from video,
\item Feature representations,
\item Predicted gesture labels,
\item Text output,
\item Speech output.
\end{itemize}

\subsection{Result Analysis}
\begin{itemize}[leftmargin=1.5em]
\item The system performs well for clear and well-defined gestures.
\item Slight variations may occur in complex or fast gestures.
\item Performance is robust under moderate background noise.
\end{itemize}
