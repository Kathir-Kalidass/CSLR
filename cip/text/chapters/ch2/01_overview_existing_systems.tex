\section{Overview of Existing Systems}
Early sign language recognition systems relied on handcrafted features and traditional machine learning algorithms such as Hidden Markov Models (HMM) and Support Vector Machines (SVM). These approaches provided limited accuracy due to their inability to capture complex visual patterns.

With the emergence of deep learning, vision-based approaches using CNNs and RNNs have become dominant. These methods significantly improve recognition accuracy by learning hierarchical feature representations directly from data.
