\section{Existing Methods and Techniques}
Modern approaches can be broadly categorized as follows:
\begin{itemize}[leftmargin=1.5em]
\item \textbf{Spatial feature-based methods:} use CNN architectures such as EfficientNet to extract hand shape and appearance features \cite{yenisari2025deep}.
\item \textbf{Temporal sequence models:} use RNN, LSTM, or GRU networks to model motion dynamics across frames.
\item \textbf{Attention-based models:} assign importance weights to relevant features and improve recognition performance.
\item \textbf{Transformer-based models:} capture long-range dependencies using self-attention \cite{zhou2024transformer}.
\item \textbf{Multimodal systems:} combine RGB images with pose or depth data for improved accuracy.
\end{itemize}
