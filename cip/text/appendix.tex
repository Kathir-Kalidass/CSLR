\section{Reproducibility Checklist}
The following checklist summarizes reproducibility controls used in this project:
\begin{itemize}[leftmargin=1.5em]
\item fixed data preprocessing sequence (decode, sample, resize, normalize),
\item explicit module boundaries between extraction, fusion, decoding, and output,
\item consistent metric definitions for recognition, sequence, and translation quality,
\item project-level scripts for environment setup and service startup,
\item integrated figure and table assets stored directly inside the thesis repository.
\end{itemize}

\section{Additional Notes on Deployment}
Report evidence indicates that deployment quality depends strongly on three practical controls: capture quality, vocabulary coverage, and signer diversity in training. For production-scale deployment, periodic data refresh and calibration checks are recommended.
